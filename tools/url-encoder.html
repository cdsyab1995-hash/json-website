URL Encoding Reference

URLs can only be sent over the Internet using the ASCII character-set. URL encoding converts characters into a format that can be transmitted over the Internet.

Space

" " → %20

Question Mark

"?" → %3F

Ampersand

"&" → %26

Equals

"=" → %3D
