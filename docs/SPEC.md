# JSON Web Tools - 功能规范 v2.0

## 项目概述
一个基于浏览器的JSON处理工具网站，采用单页面应用架构，每个功能独立一个页面，提供优秀的用户体验。

## 设计理念
- **清晰分离**：每个功能独立页面，减少认知负担
- **宽敞布局**：最大化输入/输出区域，提高工作效率
- **直观操作**：一键式操作，反馈明确
- **响应式**：完美适配桌面和移动设备

## 技术栈
- 纯HTML/CSS/JavaScript（无框架依赖）
- CSS Grid/Flexbox现代布局
- 响应式设计（移动优先）

## 核心功能模块

### 1. 首页/导航
- 简洁的顶部导航栏
- 功能卡片式展示
- 快速访问各功能模块

### 2. JSON格式化页面 (format.html)
- 大面积输入区域（占屏幕70%以上高度）
- 实时语法验证
- 格式化、校验、压缩三大功能
- 实时预览带语法高亮
- 一键复制结果

### 3. JSON转义页面 (escape.html)
- 简洁的两栏布局
- 转义/反转义切换
- 输入区和输出区分离
- 实时转换预览

### 4. JSON提取页面 (extract.html)
- JSON输入区
- JSONPath路径输入
- 提取结果展示区
- 使用示例提示

### 5. JSON对比页面 (compare.html) - 预留
- 双栏JSON对比
- 差异高亮显示

- 固定在顶部
- 高度：60px
- 背景：主色渐变
- Logo在左侧
- 导航链接在右侧

### 配色方案
- 主色：#2563eb（蓝色）
- 主色深：#1e40af
- 背景：#f8fafc（浅灰）
- 卡片背景：#ffffff
- 边框：#e2e8f0
- 成功：#22c55e（绿色）
- 错误：#ef4444（红色）
- 警告：#f59e0b（橙色）

### JSON语法高亮
- 键（Key）：#059669（绿色）
- 字符串值：#7c3aed（紫色）
- 数字值：#2563eb（蓝色）
- 布尔值：#dc2626（红色）
- null值：#64748b（灰色）

### 按钮样式
- 主按钮：蓝色背景，白色文字
- 次按钮：灰色背景，深色文字
- 成功按钮：绿色背景，白色文字
- 按钮高度：40px
- 圆角：8px
- 悬停效果：轻微上浮+阴影

## 响应式断点
- 桌面：> 1024px（双栏或全宽布局）
- 平板：768px - 1024px（单栏，间距调整）
- 手机：< 768px（单栏，全宽输入输出）
