# SEO 长尾关键词策略

## 策略核心

通过文章内容覆盖关键词，而非工具页面

## 执行计划

### 第 1-2 周（快速产出）
- 每周 2 篇博客文章
- 优先写低竞争关键词（JSON Patch、curl、Python JSON）
- 每篇 1000-2000 字，包含代码示例

### 第 3-4 周（深度内容）
- 写 3-4 篇深度技术文章（JSON Schema、GraphQL vs REST、JSON 安全）
- 开始制作配套视频脚本

### 持续（月度）
- 每月 6-8 篇新文章
- 覆盖所有工具的"使用场景"和"教程"内容
- 视频按 2 个/周节奏录制
