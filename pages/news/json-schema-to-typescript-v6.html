json-schema-to-typescript v6 Released: Generate TypeScript Types from Any JSON Schema

April 18, 2026 | 4-5 min read

The json-schema-to-typescript npm package — one of the most widely used tools in the TypeScript ecosystem — has reached version 6.0, bringing support for the latest JSON Schema specification, a rebuilt CLI, and major improvements to recursive schema handling.

Major New Features

  • JSON Schema Draft 2020-12 — full support for the latest spec including $dynamicRef, $jsonSchema, and prefix items
  • Recursive schema support — proper TypeScript type generation for self-referencing schemas (like tree structures)
  • CLI with watch mode — regenerate types automatically when schema files change
  • Banner comments — customizable headers for generated type files
  • TypeScript 5.x compatibility — uses new template literal types and infer improvements
# New CLI with watch mode
npx json-schema-to-typescript schema.json \
  --watch \
  --banner "// Generated on $(date)" \
  --out types/generated.ts

The TypeScript-JSON Schema Bridge

json-schema-to-typescript bridges two worlds: the formal contract definition (JSON Schema) and TypeScript's type system. This is particularly valuable for teams using OpenAPI specs or JSON Schema for API contracts — you can generate your TypeScript types directly from the schema, ensuring type safety end-to-end.
