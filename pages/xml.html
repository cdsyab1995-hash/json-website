XML 与 JSON 互转

在XML和JSON格式之间自由转换，支持双向转换和格式化展示

转换操作

转换规则

XML → JSON

将XML标签转换为JSON键值对，属性用@前缀，文本内容用#text表示。

JSON → XML

将JSON对象转换为XML标签，数组转换为重复标签。

自动识别

工具会自动识别输入内容是XML还是JSON格式。

常见问题

XML属性在JSON中如何表示？

XML属性会被转换为带@前缀的键。例如：

<user type="admin">张三</user>

转换为：

{"user": {"@type": "admin", "#text": "张三"}}

XML命名空间会保留吗？

会。命名空间前缀会被保留在键名中，如soap:Envelope会转换为"soap:Envelope"。

转换会丢失数据吗？

转换遵循标准映射规则，不会丢失任何数据。但部分特殊XML结构（如CDATA、注释等）可能需要额外处理。
