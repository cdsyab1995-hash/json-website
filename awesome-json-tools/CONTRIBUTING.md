# Contributing to Awesome JSON Tools

Thanks for your interest in contributing!

## How to Contribute

### Adding a New Tool

1. Make sure the tool is **free** (has a free tier is okay)
2. The tool should be **actively maintained**
3. Add it to the **appropriate category**
4. Include the **official website URL**
5. Provide a **brief description**

### Pull Request Guidelines

- Use the following format for additions:
  ```
  | [Tool Name](URL) | Description | Additional info |
  ```

- Keep descriptions **concise** (under 50 characters if possible)
- Alphabetical order within each section
- Verify all links are working

### Categories

If you're unsure which category to add to:

- **Formatters & Validators** - Tools for formatting and validating JSON syntax
- **Converters** - Tools that convert JSON to/from other formats
- **Editors & Viewers** - Visual editors and tree viewers
- **Compare & Diff** - Tools for comparing JSON documents
- **Developer Tools** - CLI tools, VS Code extensions, API clients
- **Learning Resources** - Documentation, tutorials, communities

## Reporting Issues

Found a broken link or outdated information? Please open an issue!

## Code of Conduct

Please be respectful and constructive in all interactions.
