JSONata 2.0 Launches with Native AI Query Support

April 18, 2026 | 5-6 min read

JSONata 2.0 has officially launched, and its headline feature is a first-of-its-kind capability: AI-assisted query generation. Describe what you want in plain English, and JSONata generates the corresponding JSONata expression automatically.

AI-Assisted Query Generation

JSONata has always been a powerful way to query and transform JSON data with a concise expression syntax. With 2.0, you can now describe your desired transformation in natural language, and the system generates a working JSONata expression.

// Input: natural language prompt
"What is the total revenue for orders over $100, grouped by customer?"

// Generated JSONata expression
$sum(
  $$.orders[$total > 100].{$.customer: $.total}
) ~> $each($sum) ~> $keys()

How It Works

JSONata 2.0 integrates a lightweight model specifically fine-tuned for JSONata expression generation. It understands the semantics of the JSONata language deeply, so it generates syntactically correct expressions — not approximate pseudo-code.

Traditional Query Support Remains

All existing JSONata features are preserved and improved. The AI generation is additive — you can still write expressions by hand, or use the AI feature to explore unfamiliar parts of the language.

Available Everywhere

JSONata 2.0 is available as an npm package, a REST API, a VS Code extension, and directly in the browser. The AI generation feature requires an API call to the JSONata Labs inference endpoint (free tier: 1,000 generations/month).

Whether you use JSONata for data transformation or prefer direct JSON manipulation, AI JSON Extract and AI JSON Formatter remain useful tools for working with JSON data in the browser.
