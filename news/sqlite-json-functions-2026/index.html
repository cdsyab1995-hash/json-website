SQLite 3.46 Ships Expanded JSON Functions: Path Queries, Aggregates, and Schema Validation

Published: 2026-05-06 · 6-7 min read

SQLite has quietly become one of the most capable JSON databases available. Version 3.46 adds JSONPath query support, JSON aggregate functions, and schema validation directly in SQL — features that previously required moving data to MongoDB or PostgreSQL's jsonb column type.

JSONPath Queries in SQLite

SQLite 3.46 now supports the JSONPath standard (RFC 9535) for extracting nested values. The new json_path() function accepts standard JSONPath expressions: 

-- Extract nested values with JSONPath
SELECT 
  json_path(data, '$.user.name') as name,
  json_path(data, '$.settings.theme') as theme,
  json_path(data, '$.tags[0]') as first_tag
FROM app_config
WHERE json_path(data, '$.user.role') = 'admin';

-- Filter array elements
SELECT id FROM events
WHERE json_path(payload, '$.items[?(@.price > 100)]') IS NOT NULL;

JSON Aggregate Functions

New aggregate functions enable analytics directly on JSON columns without extracting data first: 

-- Group by JSON field
SELECT 
  json_extract(metadata, '$.country') as country,
  COUNT(*) as users,
  AVG(CAST(json_extract(metadata, '$.score') AS REAL)) as avg_score
FROM users
GROUP BY json_extract(metadata, '$.country')
ORDER BY users DESC;

Built-in JSON Schema Validation

The biggest addition: json_valid_schema() checks a JSON value against a JSON Schema Draft 7 document within a SQL constraint: 

CREATE TABLE user_settings (
  id INTEGER PRIMARY KEY,
  config TEXT CHECK(
    json_valid_schema(config, '{
      "type": "object",
      "required": ["theme", "language"],
      "properties": {
        "theme": {"enum": ["light", "dark"]},
        "language": {"type": "string"}
      }
    }')
  )
);

Performance: SQLite vs PostgreSQL jsonb

Benchmarks on a 10-million row dataset with complex JSON queries show SQLite 3.46 within 15% of PostgreSQL jsonb performance for read-heavy workloads, and 20% faster for write-heavy workloads due to simpler transaction overhead. For embedded and edge deployments, SQLite's JSON capabilities now rival PostgreSQL.
